SIMULATION REQUEST

The "reuse crystal-derived assembly" protocol allows the user to take advantage of the data stored in a simulation item in the GPCRmd database in order to simplify a new simulation submission. This submission method is recomended provided that the set of crystalized components in the new simulation matches exactly the one in the reused simulation. Therefore, only small molecules belonging to the bulk environment (not crystalized components of the simulation) can be modified when using this simulation mode.

To obtain data on the set of crystalized components in a specific simulation item, the user must provide either a "Model ID" or a "Submission ID" identifier after checking the checkbox corresponding to the chosen option.

The "new submission" protocol which allows data submission from scratch is available by clicking the "Back to index" button.
