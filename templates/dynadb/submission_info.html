Pre-submission:

Before to start a submission it is necessary to do a previous preparation:

1) Prepare the files need it for the upload:

  • Coordinates. Extensions: pdb
  • Topology. Extensions: psf, top
  • Trajectory. Extensions: dcd, xtc, trr
  • Simulation parameters. Extensions: prm, prmtop, zip, tar.gz, tgz
  • Simulation protocol. Extensions: zip, tar.gz, tgz

Optionally, add other files that you consider necessary for the submission:

  • Other. Extensions: tar.gz, tgz, zip

2) Trajectories must be aligned to transmembrane part of the receptor to be submmited correctly. Also, check that the structure is located at the origin of coordinates.


3) Check that each structure in your pdb have is own ChainID and check if each of these chains have is own UniprotKB. Each chain could contain multiple segments.
