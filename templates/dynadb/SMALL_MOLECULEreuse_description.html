Step 2:
Small Molecule Information

CHECK THE SMALL MOLECULE INFO

Please, provide a detailed chemical description of the non-protein molecules needed for completing the set of components involved in the simulation to be submitted.

Note that this form contains two main blocks. The (1) summary of crystalized components block shows the description of the crystal-derived non-protein molecules involved in the GPCRmd DB item selected to be reused. In the (2) list of bulk components in the simulation block, the user must provide information about the components which have been randomly added during the simulation setup process (solvent, membrane lipids and ions whose coordinates have been ramdonly generated).

First, check that all molecules with coordinates coming from experimental data and non-protein molecules playing a crucial role in the setup of the simulations you are submitting (not bulk waters, lipids or ions randomly generated), are exactly matching the set of molecules shown in the summary of crystalized components. If so, the set of crystal-derived components retrieved by the GPCRmd DB can be reused provided that the set of proteins in the crystal to be reused (previously checked in the step 1) is exactly the same than the one the user want to submit. Otherwise, please click the ‘New Submission’ button in the initial ‘Simulation Submission’ page and enter your simulations as a new system from scratch.

After cheching the exact match between crystal-derived assemblies, add new bulk components appearing in the simulation to be submitted in the (list of bulk components in the simulation) block. Follow the next steps for every bulk component:

(A) Upload for each small molecule (non-protein) a '.sdf' or '.mol' file. A 2D chemical structure will be automatically displayed.

(B) Indicate if the uploaded structure is a co-crystalised molecule or if it belongs to bulk.

(C) Revise chemoinformatics data retrieved from the upload structure.

(D) Obtain molecule information from PubChem and ChEMBL databases.

Click the '+ Add molecule' button on the bottom of the pannel corresponding to the last uploaded molecule to add information about another molecule in your system.

Molecule data submission can be made individually by clicking the 'Submit molecule' button on the bottom of every molecule pannel or simultaneously by clicking the 'Submit all molecules' button on the bottom of the form.

For more information on this form, see the docs.
